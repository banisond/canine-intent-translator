# Canine Intent Translator

## Overview

The Canine Intent Translator is a machine learning project that explores whether canine behavioral intent can be inferred from visual and audio data.

## Goals

- Learn Python
- Learn PyTorch
- Build a labeled canine behavior dataset
- Train a behavior classifier
- Develop a prototype translator
